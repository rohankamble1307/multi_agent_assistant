# 🤖 Multi-Agent AI Assistant

An intelligent AI-powered assistant that uses multiple agents to process user queries and generate structured, real-time responses.

---

## 🚀 Overview

This project is a **Multi-Agent AI Assistant** designed to go beyond a basic chatbot.
Instead of handling queries in a single flow, it uses multiple agents working together to process user input and generate meaningful responses.

The system simulates a real-world AI pipeline where different components collaborate behind the scenes.

## 🧩 How It Works

1. User enters a query in the chat interface
2. The frontend sends the request to the FastAPI backend
3. The backend routes the query through different agents
4. Each agent processes part of the task
5. The final response is generated and returned to the user
